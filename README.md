# ⚓ Морской бой на листке в клетку

Тот самый улучшенный морской бой с задней парты: листок в клетку, линейка, карандаш —
острова, флот и добыча. Теперь в браузере и по интернету.

## Запуск

Нужна MySQL. Локально проще всего поднять её в Docker:

```bash
npm install
npm run db:up          # MySQL в Docker на порту 3307 (том sb-mysql-data — данные сохраняются)
cp .env.example .env   # уже указывает на этот контейнер; правь под свою БД при желании
npm run dev            # http://localhost:3456, авто-перезапуск при правках (.env подхватится сам)
```

`npm run db:down` — убрать контейнер (том с данными остаётся). Можно указать любую другую
MySQL — впиши свой `DATABASE_URL` в `.env`. Порт сервера меняется переменной `PORT`.
На проде (Render и т.п.) `.env` не нужен — переменные задаются в панели (см. `DEPLOY.md`).

## Как играть

1. На главной вписываешь ник, выбираешь число игроков (2–4) и таймер на ход
   (без таймера / 1 / 2 / 5 минут) → «Создать баттл».
   Можно и наоборот — кнопка **«👥 найти игру»** на главной открывает список
   открытых лобби (обновляется в реальном времени); жми «Войти» и попадёшь в чужой баттл.
2. Получаешь ссылку вида `/game/XXXXXXX` — кидаешь друзьям. Они открывают её,
   вписывают ник и попадают в лобби.
3. Когда собрались (минимум 2), создатель жмёт «Начать баттл» — генерируется карта:
   базы-острова по краям, лут-острова и рыбные места в центре.
4. Ходят по очереди. За ход — **одно** действие:
   - **Купить** корабли на верфи (появляются у твоего порта);
   - **Собрать добычу**: любой корабль лутает остров с кладом, если дотягивается
     (рыбалка отдельного действия не требует — см. ниже);
   - **Передвинуть один корабль** «по линейке» — клик по кораблю → Плыть → точка
     в круге дальности;
   - **Выстрелить**: клик по кораблю → Стрелять → вражеский корабль или порт
     в круге дальности стрельбы.
5. Потопил корабль — лутаешь золото с обломков. Разбил порт — соперник выбывает,
   а ты забираешь половину его казны. Последний живой порт побеждает. 👑
   Порт очень прочный (840 HP), приносит немного золота каждый ход и **огрызается** 🏰
   по атакующему его кораблю (−15 HP, обломки достаются защитнику): лёгким кораблём
   в одиночку базу не взять, да и линкору нужно несколько залпов — эффективнее
   давить несколькими кораблями.
   - **🐟 Рыбалка пассивная**: баркас, стоящий в рыбном месте, сам приносит золото
     (+8) в начале каждого твоего хода — действие на это не тратится.
6. По морю бродят нейтральные **пираты** 🏴‍☠️: медленно дрейфуют между ходами,
   могут исчезать в тумане и появляться вновь. Потопи — забери награду (150–350 золота).
   Изредка приходит жирный **👑-босс** (220 HP) с большим кушем — 400–800 золота.
7. Соперник пропал? **«📯 Поторопить»** — ему уйдёт письмо (если входил через Google)
   и останется 10 минут на ход, иначе автопропуск. Сам безнадёжно застрял —
   жми **«🏳️ Сдаться»**; из лобби можно просто выйти.

Игра живёт по ссылке: можно закрыть вкладку и вернуться позже — состояние хранится
в MySQL и переживает рестарт сервера и деплой.

## Флот

| Корабль          | Цена | HP  | Урон | Дальность огня | Ход | Особое |
|------------------|------|-----|------|----------------|-----|--------|
| Рыбацкий баркас  | 60   | 30  | 5    | 70             | 140 | 🐟 +8 золота каждый ход в рыбном месте (пассивно) |
| Шхуна            | 110  | 60  | 15   | 110            | 170 | стартовый корабль ×2 |
| Бриг             | 220  | 110 | 28   | 140            | 135 | |
| Фрегат           | 380  | 170 | 42   | 165            | 110 | стартовый «крейсер»; 💥 бортовой залп |
| Линкор           | 500  | 280 | 65   | 190            | 90  | 🏰 урон по порту ×1.5 (≈98); 💥 бортовой залп |

**💥 Бортовой залп** — тяжёлые корабли (фрегат, линкор) могут вместо одиночного
выстрела дать залп по **всем вражеским боевым кораблям** в радиусе разом (включая
пиратов), но каждая цель получает на 20% меньше урона. Рыбацкие баркасы залп не
трогает. Доступно, когда в радиусе 2+ боевых врага — анти-сварм против стаи.

Расстояния в игровых единицах, 1 клетка = 40. Стартовый капитал — 250 золота,
HP порта — 840.

## Игра по интернету

Сервер один на всех, ссылка ведёт на конкретную сессию. Чтобы друзья зашли извне:
- проще всего — туннель: `npx localtunnel --port 3456` или `cloudflared tunnel --url http://localhost:3456`;
- или задеплоить на любой VPS/хостинг с Node.js (нужен один процесс, БД — файл рядом).

## Необязательные настройки (переменные окружения)

| Переменная | Что включает |
|---|---|
| `GOOGLE_CLIENT_ID` | Кнопку «Войти через Google» (ник остаётся свой, почта — для уведомлений) |
| `SMTP_HOST`, `SMTP_PORT`, `SMTP_USER`, `SMTP_PASS`, `SMTP_FROM` | Письма «твой ход, 10 минут» при нажатии «Поторопить» |
| `BASE_URL` | Адрес сайта для ссылок в письмах (например `https://game.example.ua`) |
| `PORT` | Порт сервера (по умолчанию 3456) |

Без этих переменных всё работает: вход остаётся гостевым (ник + токен в браузере),
а «Поторопить» просто включает 10-минутный таймер без письма.

## Технологии

- **Сервер**: Node.js + Express + Socket.IO (вся логика и валидация ходов на сервере).
- **БД**: MySQL (mysql2) — игроки, сессии игр, результаты, лидерборд. Подключение через
  `DATABASE_URL` или `DB_HOST/DB_USER/DB_PASSWORD/DB_NAME` (см. DEPLOY.md).
- **Клиент**: ванильный JS + Canvas, без сборки. Личность игрока — ник + токен
  в localStorage (без регистрации).
